# Class 5: MyBlocks (Functions)
## Class 5: My Blocks / Functions

<!-- type: lesson -->

Class 5: My Blocks / Functions

### **Unlock Superpowers with MyBlocks on SPIKE Prime!**

![mblox](https://dp4rp4tcpuoo1.cloudfront.net/R400V2/mblox.png)

Using **MyBlocks** in SPIKE Prime is like giving your robot **special abilities** that can be used over and over! They help you:


- 
**Save time** – Write code once and reuse it whenever needed.

- 
**Stay organized** – Break long programs into clear, easy-to-read blocks.

- 
**Work smarter** – Share MyBlocks with teammates or apply them in different projects.


With MyBlocks, you’ll **code faster, stay creative, and improve your programming skills**. It’s a powerful way to make your robot more efficient!

In programming, MyBlocks are similar to **functions**—they group instructions into reusable blocks, so you don’t have to rewrite the same steps multiple times. Just call the function when needed! Like MyBlocks, functions keep code **clean, organized, and easy to manage**—a skill that applies to any programming language.
